# Final year project in robotics with Louise (the beacon) and Waldo (the robot)
This Eurobot 2024 project, based on a Mars exploration theme, focused on developing a ROS-based robotic system enabling autonomous decision-making in a competitive multi-robot environment. A central camera system was used to detect and locate key game elements such as plants, ArUco markers, and solar panels, extracting their positions (x, y), orientations (θ), and identifiers. The processed data was then published through a ROS architecture to distributed robotic nodes, allowing real-time communication and global environment awareness for each robot. This ROS2-based system improved coordination, perception, and autonomous task execution within a structured robotic framework.
